# Fastener Selection Checklist (Engineering + Sourcing)

Use this checklist to reduce rework, quality risk, and communication gaps across teams.

## 1) Joint Design and Loading

- Base materials (steel, aluminum, composites, etc.)
- Load mode (tension, shear, vibration, impact)
- Preload target and anti-loosening requirements

## 2) Standard and Dimensional Definition

- Standard system (DIN / ISO / ANSI / ASTM / JIS)
- Diameter, length, pitch, tolerance
- Compatibility of bolt, nut, and washer set

## 3) Material and Strength Grade

- Material choice (carbon steel, alloy steel, stainless steel)
- Property class (4.8 / 8.8 / 10.9 / 12.9 or A2-70 / A4-80)
- Expected failure modes (yield, fatigue, corrosion, loosening)

## 4) Surface Treatment and Environment

- Service condition (indoor, outdoor, coastal, chemical, high-temp)
- Coating system (zinc, HDG, Dacromet, Ruspert, etc.)
- Verification method (salt spray, torque window, field feedback)

## 5) Quality and Delivery Control

- Batch traceability package (MTC, inspection reports)
- Incoming and in-process sampling plan
- Export packaging and logistics protection

## 6) For Custom/OEM Projects

- Drawing revision and version control
- Prototype confirmation and engineering change records
- Pre-production risk review (PFMEA when required)
