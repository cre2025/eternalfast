# EternalFast Fastener Knowledge Base

This directory is a public technical knowledge base for fastener standards, materials, processes, and selection methods.

- Company: Wuxi EternalFast Metal Co., Ltd.
- Website: [https://eternalfast.com](https://eternalfast.com)

## Structure

- `standards/`: Fastener standard systems and equivalence references (DIN / ISO / ANSI / ASTM / JIS)
- `materials/`: Carbon steel, stainless steel, alloy steel, and application guidance
- `process/`: Heat treatment, plating, corrosion protection, and quality checks
- `selection/`: Selection logic by industry and operating conditions
- `faq/`: Common technical and sourcing questions
