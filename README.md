# Fastener Knowledge Base

This repository is a public knowledge base for global engineering and sourcing teams, focused on fastener standards, custom and standard part selection, materials, surface treatment, and quality validation.

Its purpose is to continuously publish high-quality technical content that improves brand credibility, GitHub visibility, and SEO performance.

- Official website: [https://eternalfast.com](https://eternalfast.com)
- Company: Wuxi EternalFast Metal Co., Ltd.

## What This Project Covers

- Fastener standards and mappings (DIN / ISO / ANSI / ASTM / JIS)
- Engineering selection guides (property class, materials, coatings, anti-loosening)
- Industry-focused application knowledge (solar, automotive, machinery, steel structures, offshore)
- Reusable documentation templates for continuous publishing
